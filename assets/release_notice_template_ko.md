## 3) 설치 방법

1. 최신 ZIP 파일을 다운로드합니다.
2. 원하는 위치에 압축을 풉니다.
3. `{실행파일명}`을 실행합니다.
4. 게임 폴더가 자동으로 잡히지 않으면 [찾아보기] 또는 경로 선택으로 설치 폴더를 지정합니다.
5. [한국어 패치 설치] 또는 설치 메뉴를 실행합니다.
6. 완료 후 게임을 다시 실행해 한글 적용 여부를 확인합니다.

경로 확인 팁:

- Steam: 라이브러리 -> 게임 우클릭 -> 관리 -> 로컬 파일 보기
- Xbox PC/Game Pass: Xbox 앱 또는 Windows 앱 설정에서 설치 위치를 확인

## 4) 제거(원본 복구) 방법

1. `{실행파일명}`을 실행합니다.
2. [원본 복원] 또는 복원 메뉴를 실행합니다.

## 6) 자동으로 진행되는 작업

설치 시 아래 작업이 자동으로 진행됩니다.

- 게임 설치 경로 확인
- 지원 플랫폼/게임 버전 확인
- 원본 파일 백업 생성
- 한국어 텍스트 적용
- 필요한 폰트/런타임/플러그인 설치
- 패치 적용 후 검증

패치 파일에는 완성된 게임 원본 파일 전체가 들어 있지 않습니다.

사용자 PC에 설치된 원본 게임 파일에 한글 텍스트를 적용하는 방식입니다.

## 8) 플레이테스트 부탁드립니다

실제 플레이 중 아래 유형을 발견하면 제보해 주세요.

- 문맥이 어색한 번역
- 말투/호칭 이탈
- UI 줄넘침 또는 잘림
- 영어 잔존
- 내부 키/태그 노출
- 오타/오번역
- 고유명사 표기 흔들림
- 설치 실패 또는 패처 오류
- 게임 업데이트 이후 패치가 적용되지 않는 문제

제보할 때 같이 알려주시면 좋은 정보:

- 플랫폼
- 패치 버전
- 게임 버전/buildid
- 문제가 나온 화면 스크린샷
- 가능하면 원문 또는 현재 표시되는 문장
- 설치 중 오류가 났다면 패처 로그 또는 오류 메시지
